CSS3背景

英文原版在这里

通过上一节的学习，你可能已经熟悉了CSS3边框。在本节中，我们将要学习所有主流浏览器都支持的CSS3背景。

CSS3的系列background属性用于调整背景各项属性的大小。它还可以用于实现多重背景。在CSS3之前，我们无法调整背景大小，现在有了CSS3就可以实现这一功能了。

背景主要有两个属性，可用于背景确定，背景大小，多背景图等。下面两个就是CSS3背景属性：

✔ background-size

✔ background-origin

    ✔ background-origin:border-box

    ✔ background-origin:content-box

background-size属性用于确定背景大小，可以是像素格式或百分比格式。

示例与语法

浏览器支持情况

					
<!DOCTYPE html> 
<html>     
    <head>
         <title>标题</title>
    </head>     
    <style>         
        #background_size
        {
            background:url(./img/flower.png);
            background-size:100px 40px;
            -moz-background-size:100px 40px; /* Firefox 3.6 */
            -webkit-background-size:100px 40px;
            background-repeat:no-repeat;
            padding-top:40px;
        }         
    </style>     
    <body> 
        <div id="background_size"> </div>         
        <div>
            <p>以下是原图：</p>
            <img src="./img/flower.png" alt="image" title="original image"/>
        </div> 
    </body>     
</html>
