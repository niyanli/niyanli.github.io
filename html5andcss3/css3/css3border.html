CSS3边框

英文原版在这里

在上一节中，学习了关于CSS3动画的基础知识。 在本节中，我们将学习CSS3边框，当你尝试为你的内容设计一个时尚的容器时这是最有用的。

CSS3边框减少了Photoshop和其他图形应用程序的人力成本，个人就可以在CSS3边框的帮助下创建圆形边框，边框阴影，基于图像的边框等。

基本上我们使用三个属性来创建边框：

✔ border-radius

✔ box-shadow

✔ border-image

border-radius是CSS3的一个属性，通过它我们可以创建圆角。

box-shadow是CSS3的一个属性，通过它我们可以创建边框的阴影。

border-image是CSS3的一个属性，通过它我们可以创建自定义边框，因为我们可以将自己的图像作为边框。

示例与语法

使用border-radius属性可以做出圆角边框

使用box-shadow属性可以给一个盒子创建边框阴影

你可以看到自定义边框。图片可以是*.png或者*.jpg格式

浏览器支持情况

					
<!DOCTYPE html> 
<html>     
    <head>
         <title>标题</title>
    </head>     
    <style>    	
        #border_radius
        {
            border:2px solid;
            font-size: 14px;
            color: #ffffff;
            font-weight: bold;
            padding: 10px;
            background: #6AAFCF;
            border-radius:25px;
            -moz-border-radius:25px; /* For Firefox Browser */
            -webkit-border-radius: 25px; /* For Safari and Google Chrome Browser */
            -o-border-radius: 25px /* For Opera Browser */
        }
 
        #box_shadow
        {
            font-size: 14px;
            color: #ffffff;
            font-weight: bold;
            padding: 10px;
            background: #6AAFCF;
            -moz-box-shadow: 15px 15px 5px #888245; /* For Firefox/Mozilla */
            -webkit-box-shadow: 15px 15px 5px #888245; /* For Google Chrome and Safari */
            -o-box-shadow: 15px 15px 5px #888245; /* For Opera */
            box-shadow: 15px 15px 5px #888245;
        }
         
        #border_image
        {
            border:15px solid transparent;
            -moz-border-image:url(./img/border.png) 30 30 round; /* Firefox */
            -webkit-border-image:url(./img/border.png) 30 30 round; /* Safari and Chrome */
            -o-border-image:url(./img/border.png) 30 30 round; /* Opera */
            border-image:url(./img/border.png) 30 30 round;
        }               
 
    </style>     
    <body> 
        <p id="border_radius">
        	使用border-radius属性可以做出圆角边框            
        </p>         
        <p id="box_shadow">
        	使用box-shadow属性可以给一个盒子创建边框阴影
        </p>
        <p id="border_image">
        	你可以看到自定义边框。图片可以是*.png或者*.jpg格式
        </p>
    </body>
</html>
