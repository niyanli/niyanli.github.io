CSS3动画

英文原版在这里

现在先从CSS3动画开始学习。动画是CSS3最具创新的特性，虽然它还没有被所有主流浏览器支持，但是它很适合减少外部插件的情况。

CSS3动画没有使用Flash或任何其他动画应用程序而使对象动画化。使用CSS3的这一功能，你可以通过动画方式将对象从一种样式转换为另一种样式。

除IE之外的所有主流浏览器都支持动画功能。它将对象的样式逐渐地更改为另一种样式，完整的动画依赖于关键帧（keyframes）的声明。

浏览器支持情况

					
<!DOCTYPE html>
<html>
    <head>
    	 <title>标题</title>
    </head>
	<style>
		#animated_div
		{
			text-align:center;
			width:60px;
			height:40px;
			background:#6699FF;
			color:#ffffff;
			position:relative;

			padding:10px;
			animation:animated_div 5s infinite;
			-moz-animation:animated_div 5s infinite;
			-webkit-animation:animated_div 5s infinite;
			border-radius:5px;
			-webkit-border-radius:5px;
		}
		 
		@keyframes animated_div
		{
			0%   {left:0px;}
			20%  {left:50px; background-color: green;}
			40%  {left:140px; background-color: red;}
			60%  {left:280px; background-color: yellow;}
			80%  {left:425px; background-color: blue;}
			100% {left:0px; background-color: pink;}
		}
		 
		@-webkit-keyframes animated_div
		{
			0%   {left:0px;}
			20%  {left:50px; background-color: green; margin-top: 50px;}
			40%  {left:140px; background-color: red;margin-top: 0px;}
			60%  {left:280px; background-color: yellow;margin-top: 50px;}
			80%  {left:425px; background-color: blue;margin-top: 0px;}
			100% {left:0px; background-color: pink;}
		}
		 
		@-moz-keyframes animated_div
		{
			0%   {left:0px;}
			20%  {left:50px; background-color: green;}
			40%  {left:140px; background-color: red;}
			60%  {left:280px; background-color: yellow;}
			80%  {left:425px; background-color: blue;}
			100% {left:0px; background-color: pink;}
		}
	</style>
    
    <body>
    	<div id="animated_div">
    		动画div
    	</div>
    </body>
</html>
